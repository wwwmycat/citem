你好，未来！

2017年12月19日

自F1诞生以来，AMG工程师们就一直梦想着将创造过无数赛道传奇的尖端科技引入街头。终于，在连续3年斩获F1车队与车手双料冠军后， 这一梦想成为现实。双座版梅赛德斯-AMG Project ONE将最新、最高效的F1赛车混合动力技术悉数运用于公路之上，使赛道与公路接轨！ 这台混血猛兽可瞬时迸发出超过1000马力的强劲动力，顷刻尽揽350公里/小时最高速度，在6秒内即可完成0-200公里/小时的加速过程。梅赛德斯-AMG Project ONE将令人叹服的赛道性能与无可匹敌的F1混合动力技术完美融合，这不仅是向AMG在赛车运动中所取得的无数荣耀致敬，亦是“非凡驾驭”对未来的强势声明！

性能至上
形式遵循功能，是在梅赛德斯-AMG Project ONE的设计中始终贯穿的原则，每个细节均为车辆的整体性能做出独特贡献：车身的肌肉线条围绕中后置发动机而设计； 前进感十足的驾驶舱外形、宽大的轮毂和带有空气动力学套件的纤细腰身，都明确地宣告出其源于赛道的独特身份。与车身同宽的进气口，搭配宽大的黑色前扰流板，营造出车辆前部凶猛无比的视觉感受，而扁平LED头灯则无缝融入车身轮廓。车顶处的设有空气入口，与垂直鲨鱼鳍扰流板相连，凌厉动感油然而生。车辆尾部带有强烈的F1赛车风格，呈一大两小圆形的中置排气管、极富力量感的尾部扩散器及主动式双层尾翼，这些高度性能导向的设计赋予了车辆前所未有的鲜明特点。

当纯粹优雅遇到尖端科技
基于性能和实用性的完美融合，梅赛德斯-AMG Project ONE无论是否身处赛道， 它的的内部设计均可提供纯粹的F1体验，其所选用颜色与材料配置的灵感亦源于F1赛车。配备双悬浮式液晶中控屏幕的驾驶舱，内部采用大面积创新材质并呈现出令人着迷的未来主义色彩。整个内部轮廓基于人体工程学打造：车辆踏板及F1风格方向盘可根据驾驶需求进行调整；中央区的隆起为碳纤维支撑结构的一个功能性组成部分，在视觉上分割出驾驶者与乘客的各自区域，并巧妙地融入了车身单体式设计；独特形制的一体式的桶椅设计让人产生对于赛车世界的无限遐想；座椅表面以灰褐色nappa皮革点缀其间，并饰以AMG独具特色的黄色缝线。

超凡性能
梅赛德斯-AMG Project ONE所搭载的EQ Power+ 高性能插入式混合动力驱动系统，完全移植于F1赛车，经过与梅赛德斯-AMG高性能动力系统研发中心专家们的通力合作，终于得以实现。该系统由高度集成、且网络智能化的1.6升6缸涡轮增压发动机及4台电动机组成：其中1台电动机被集成在涡轮增压器中，1台直接安装在发动机上，并与曲轴箱、以及剩余两个位于前轮的电动机相连。
